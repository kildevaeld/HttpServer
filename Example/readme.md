### Example

A simple console application :

Usage:

  -c, --config     

  -p, --port       (Default: 8080) 

  -v, --verbose    (Default: False) 

  -r, --root       

  -l, --log        (Default: False) 

  --help           Display this help screen.


The config options takes a path to a json file. 
Valid fields are:  port and root
